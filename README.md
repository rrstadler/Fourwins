# 4 Wins (Connect Four)

A browser-based Connect Four game — no installation required.

## How to play

Open `index.html` in any modern browser and click a column to drop your disc.
Connect **4 discs** in a row (horizontally, vertically, or diagonally) to win.

## Features

- Human (Red) vs Computer (Yellow)
- Three AI difficulty levels:
  - **Easy** — random moves
  - **Medium** — minimax search, depth 3
  - **Hard** — minimax + alpha-beta pruning, depth 7
- Drop animation with bounce effect
- Winning discs highlighted with a pulsing glow
- Cumulative score tracking across games
- New Game button; changing difficulty auto-restarts

## Tech stack

Pure HTML / CSS / JavaScript — single file, zero dependencies.
